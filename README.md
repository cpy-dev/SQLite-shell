# SQLite-shell
### Lightweight cross platform open source GUI SQLite shell

# Installation

Download the tar.gz file

    $ 

# Usage
    
    $ sqliteshell
     
You can add the executable to /bin folder, to execute it from any folder.

    $ cp sqliteshell /bin
